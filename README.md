# opslego
面向个人卖家和小微商家的轻量级电商运营管理平台前端
# 运营乐高 (OpsLego) - 电商运营自动化平台

## 🚀 在线演示

[https://opslego.muyang-bai.com](https://opslego.muyang-bai.com)

## ✨ 功能亮点

- 📱 **全端适配** - PC/平板/手机三端完美响应式
- 🤖 **智能客服** - 7×24小时AI自动回复
- 📊 **数据雷达** - 实时监控竞品价格与销量
- 🖼️ **AI内容生成** - 自动化商品标题与文案创作
- 📦 **商品管理** - 自动擦亮、批量上架、多店铺货
- 🔐 **安全防护** - 数学题验证 + 邮箱验证码双重防护

## 🛠️ 技术栈

- **前端框架**: Vue.js 3 + Vite
- **UI 组件库**: Element Plus
- **CSS 框架**: Tailwind CSS
- **状态管理**: Pinia
- **图表引擎**: ECharts
- **HTTP 客户端**: Axios

⚠️ 注意
本仓库仅包含前端展示层代码。
后端核心业务逻辑、API接口、自动化引擎等模块未开源。

## 📦 本地运行

```bash
# 克隆仓库
git clone https://github.com/muyang-bai/opslego.git
cd opslego/client

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
